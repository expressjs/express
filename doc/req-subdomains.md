<h3 id='req.subdomains'>req.subdomains</h3>

An array of subdomains in the domain name of the request.

{% highlight js %}
// Host: "tobi.ferrets.example.com"
req.subdomains
// => ["ferrets", "tobi"]
{% endhighlight %}
